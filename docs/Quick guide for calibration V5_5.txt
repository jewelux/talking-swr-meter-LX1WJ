Quick guide for calibration of the Talking SWR Meter LX1WJ V5_5

- Use a good reference SWR and power meter.
- Read power and SWR directly from the reference instrument.
- Enter these values directly into the calibration table of the Talking SWR Meter LX1WJ.
- Pfwd [W] = real measured power
- SWR read = measured SWR
- Capture Ufwd [V] and Uref [V] with the Measure button when tuning.
  These are the voltages measured by the microcontroller.
- Save the calibration table and export it if needed.
